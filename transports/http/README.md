# go.loglayer.dev/transports/http

Generic batched HTTP POST transport for LogLayer. Pluggable encoder, async
worker, configurable batching and retries. Use it directly to talk to any
log-ingestion API; the Datadog transport is built on top of it.

## Install

```sh
go get go.loglayer.dev/transports/http
```
